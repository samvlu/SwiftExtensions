# UIColorHex
Use hexidecmal values with UIColor

## Install & Usage
1. drag and drop "UIColorHex.swift" into project folder
2. type hexidecmal value after "0x" in UIColor()  

Example:
``` swift
sampleTitle.textColor = UIColor(hex: 0x1B6FEF)
```

### Requirements
Swift 2.0  
iOS 9.0 or above
